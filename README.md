# 🤖 AI Salon Assistant

An AI-powered Salon Booking System that allows customers to book salon appointments through an intelligent chatbot while providing salon owners with an easy-to-use admin dashboard.

## ✨ Features

- 🤖 AI-powered chatbot for booking appointments
- 💇 Multiple salon services
- 📅 Appointment scheduling
- 📱 Responsive UI
- 📊 Admin Dashboard
- 🔍 Customer Search
- ✅ Booking Status Management

---

## 🛠️ Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- JSON / SQLite

**AI**
- Google Gemini API

---

## 🚀 How to Run

```bash
npm install

node server.js
```

Open:

```
http://localhost:3000
```
